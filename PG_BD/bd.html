Que es una base de datos?

Una base de datos es un conjunto de datos organizados que se almacenan y gestionan de manera estructurada.

Para que sirve una BD?

Una base de datos sirve para almacenar, organizar y gestionar información de manera eficiente y segura.

que es un campo?

Un campo es una columna en una tabla de base de datos que almacena un tipo específico de información.

que es un registro?

Un registro es una fila en una tabla de base de datos que contiene información sobre un objeto específico.

que es un dato?

Un dato es una unidad de información que se almacena en una base de datos.

que es un diagrama entidad relacion?

Un diagrama entidad-relación es una representación gráfica que muestra las entidades y las relaciones entre ellas en una base de datos.
